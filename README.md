# Tic Tac Toe Game 🎮

An interactive Tic Tac Toe game built using HTML, CSS, and JavaScript.

## 🚀 Features
- Two-player gameplay (X vs O)
- Winner detection logic
- Reset & New Game functionality
- Responsive UI using Flexbox and vmin units

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript (DOM Manipulation)

## 📌 How to Play
1. Player O starts first
2. Players take turns
3. First to align 3 wins

## 🎯 Future Improvements
- Draw detection
- Winning highlight
- Animations
